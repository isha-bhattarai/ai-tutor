# AI Academic Tutor 🎓

A console-based AI-powered academic tutor built with Python and Google Gemini API.
Built as a university project for Pokhara University.

## Built With
- Python 3.x
- Google Gemini API (google-genai)
- LangChain (coming in Steps 6-7)

## Setup Instructions
1. Clone this repository
2. Install dependencies:
pip install -r requirements.txt
3. Create a `.env` file and add your Gemini API key:
GEMINI_API_KEY=your_key_here
4. Run any step:
python step1.py
